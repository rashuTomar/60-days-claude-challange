# 📰 Media Integrity Analyzer

An interactive single-page web application that teaches **media literacy** through guided discovery instead of traditional quizzes.

Rather than testing what users already know, the application encourages them to observe, think critically, make decisions, and then reveal explanations that demonstrate how headlines and social media content can influence perception.

Everything runs completely offline in a **single HTML file** using only **HTML, CSS, and Vanilla JavaScript**.

---

## ✨ Features

### 🎨 Theme Selection

* Multiple built-in color themes
* Includes **Claude Orange** theme
* Instant theme switching

### 📰 Headline Detective

Learn how sensational headlines influence readers.

Users can:

* Read a fictional news headline and article
* Decide whether they would click it
* Identify exaggerated or misleading wording
* Reveal:

  * Headline Accuracy Score
  * Highlighted mismatches
  * Detailed explanation
  * Fair rewritten headline
  * Key media literacy takeaway

---

### 🎭 Emotion Detector

Understand how emotional language shapes reactions.

Users can:

* Read a fictional social media post or article excerpt
* Record how it made them feel
* Identify emotional trigger words
* Reveal:

  * Intended audience
  * Desired emotional response
  * Manipulation technique
  * Highlighted emotional phrases
  * Neutral rewritten version
  * Key takeaway

---

### 📊 Live Media Integrity Metrics

The application continuously updates:

* Headline Accuracy
* Source Reliability
* Emotional Manipulation
* Audience Targeting

These indicators help users understand how different aspects contribute to the credibility of media content.

---

### 🏆 Media Integrity Dashboard

After completing all challenges, users receive a personalized dashboard showing:

* Overall Media Integrity Score
* Concepts learned
* Biggest media red flag
* Three practical media literacy habits
* Replay option with completely new scenarios

## 🚀 Getting Started

1. Download the HTML file.
2. Open it in any modern web browser.
3. Choose your preferred theme.
4. Complete the interactive media literacy challenges.
5. Review your Media Integrity Dashboard.
6. Replay with new scenarios to continue learning.

No installation or internet connection is required.

---

## 🎓 Educational Focus

Unlike traditional quizzes, this project follows a **learn-by-discovery** approach.

Each challenge begins by explaining:

* What the concept is
* Why it matters
* How it applies to everyday media consumption

Users first make their own observations before revealing explanations, encouraging active learning and long-term understanding.
