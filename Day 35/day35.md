🧩 Prompt Puzzle — Master AI Prompting Through Play

An interactive, game-based web application designed to help users learn AI Prompt Engineering through hands-on challenges instead of passive reading.

Players solve prompt puzzles, compare AI outputs, optimize prompts, and receive detailed performance feedback—all inside a single offline HTML file.

🎯 Overview

Prompt engineering is one of the most valuable AI skills today, but most learning resources are theoretical.

Prompt Puzzle transforms learning into an engaging game where players build, improve, and evaluate prompts across different domains while receiving instant feedback.

The application is beginner-friendly while remaining challenging for advanced users through adjustable difficulty levels and randomized scenarios.

✨ Features
🎮 Interactive Gameplay
Three unique challenge modes

Domain-specific prompt exercises
Replay with new scenarios
🧩 Challenge Types
1. Build the Prompt

Arrange prompt blocks into the best possible prompt.

2. Clean the Prompt

Remove unnecessary or redundant prompt instructions while keeping the prompt effective.

3. Choose the Best Prompt

Compare multiple prompts and identify the most effective one based on prompt engineering principles.
