# 🏥 Hospital Admission Readiness Simulator

An interactive healthcare workflow simulator that models the hospital admission readiness process. Users take on the role of a **Hospital Admission Coordinator** and complete administrative tasks required before a patient can be admitted.

> **Disclaimer:** All provider names, payer names, and healthcare data used in this simulator are **illustrative training data** and are intended solely for educational purposes.

## Features

* Patient admission setup for multiple clinical scenarios
* Multiple admission types (Inpatient, Observation, Emergency, ICU, Same-Day Surgery)
* Prior Authorization workflow with approval, pending, denial, and appeal paths
* Insurance verification and eligibility checks
* Bed assignment workflow
* Clinical documentation and physician order validation
* Consent completion tracking
* Admission readiness score based on weighted criteria
* Interactive admission timeline
* Care coordination cards for key hospital teams
* Risk tracking across documentation, insurance, bed, and clinical readiness
* Governance snapshot with healthcare industry benchmark estimates
* Final admission decision with readiness summary

## Clinical Scenarios

* Acute Myocardial Infarction (Acute MI)
* Congestive Heart Failure (CHF)
* Pneumonia
* Elective Surgery
* Hip Fracture

## Workflow

1. Configure admission details
2. Analyze admission readiness
3. Complete administrative workflow tasks
4. Improve readiness score
5. Review risks and governance insights
6. Receive the final admission decision

## Technologies Used

* HTML5
* Tailwind CSS (CDN)
* Vanilla JavaScript

## Learning Outcomes

* Hospital admission workflow
* Prior Authorization process
* Insurance verification
* Clinical documentation requirements
* Observation vs Inpatient admissions
* CMS 2-Midnight Rule
* Utilization Review concepts
* InterQual & Milliman medical necessity criteria
* Care coordination in hospital operations

## Purpose

This project was built as part of my **#60DaysOfAI** challenge to better understand healthcare administration workflows through interactive AI-assisted simulations.

**Note:** This application is intended for learning and demonstration purposes only and should not be used for real clinical or administrative decision-making.
