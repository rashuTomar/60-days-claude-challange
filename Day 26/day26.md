# 🏥 Prior Authorization Workflow Simulator

An interactive **Prior Authorization (PA) Workflow Simulator** built as a **single self-contained HTML file** using HTML, CSS, and Vanilla JavaScript. The application gamifies the US healthcare prior authorization process through a visual drag-and-drop workflow, helping users understand how patients, providers, and payers interact during authorization requests.

## ✨ Features

* Interactive drag-and-drop workflow
* Three workflow lanes:

  * Patient
  * Provider
  * Payer
* Multiple patient scenarios:

  * Elective Surgery
  * MRI
  * Specialty Medication
  * Inpatient Admission
* Medical necessity evaluation
* Prior Authorization document collection
* Submission to payer
* Multiple review outcomes:

  * Approval
  * Pend
  * Denial
  * Appeal
  * Peer-to-Peer Review
* Educational explanations after every workflow step
* Progress tracker
* Days elapsed counter
* Efficiency score
* Approval celebration animation
* Workflow summary screen
* Restart / New Patient functionality
* Fully responsive modern UI

## 🛠️ Tech Stack

* HTML5
* CSS3
* Vanilla JavaScript

## 🎮 Workflow Simulation

Users guide a patient case through the complete Prior Authorization lifecycle by interacting with different workflow stages, collecting required documentation, submitting requests, and handling payer decisions until the case reaches its final outcome.

## 📂 Project Structure

```text
Prior-Authorization-Workflow-Simulator/
│── index.html
```

## 🚀 How to Run

1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. Select a patient scenario.
4. Progress through the workflow using drag-and-drop interactions.
5. Complete the authorization process and review the final summary.
