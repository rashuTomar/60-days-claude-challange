# Day 20 – Face Puzzle Game

## Overview

Built an interactive **Face Puzzle Game** using Claude, where users can capture their face using the device camera and solve it as a customizable jigsaw puzzle. The project demonstrates the integration of browser APIs, canvas-based image processing, drag-and-drop interactions, and persistent game statistics.

## Features

* Live webcam access with photo capture
* Multiple difficulty levels (3×3, 4×4, 5×5)
* Dynamic puzzle generation from captured image
* Mouse and touch drag-and-drop support
* Live timer, move counter, and progress tracker
* Automatic win detection
* Local leaderboard using LocalStorage
* Responsive design for desktop and mobile

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Canvas API
* MediaDevices API (`getUserMedia`)
* LocalStorage API

## Key Learnings

* Implemented webcam access using the MediaDevices API.
* Processed images with the Canvas API to generate puzzle pieces.
* Developed drag-and-drop mechanics for desktop and touch devices.
* Managed game state, timers, and move tracking efficiently.
* Stored and displayed persistent leaderboard data with LocalStorage.
* Improved responsive UI design for cross-device compatibility.
* Leveraged Claude to accelerate development and solve complex frontend challenges.

## Outcome

Successfully built a fully functional browser-based Face Puzzle Game while gaining hands-on experience with browser APIs, image manipulation, interactive UI development, and AI-assisted software engineering.
