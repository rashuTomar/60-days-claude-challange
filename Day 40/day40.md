# AI Assistant Builder

An interactive AI-powered application that helps users design and generate their own custom AI assistants through a guided interview process.

Instead of asking users to write long prompts manually, the application conducts a structured interview, understands the assistant's purpose, and automatically creates a production-ready system prompt, a tailored interface, and implementation documentation.

---

## ✨ Features

* Interactive step-by-step assistant creation workflow
* Multiple-choice interview experience
* Domain and niche selection
* User audience and outcome definition
* Input and output customization
* Personality and tone selection
* Automatic generation of a production-quality system prompt
* Premium, purpose-built responsive interface
* Live Claude API integration
* Loading, error, and empty state handling
* Built-in documentation panel explaining the implementation
* Single self-contained HTML file (no external libraries)

---

## 🛠️ Tech Stack

* HTML5
* CSS3

---

## 🚀 How It Works

1. Launch the application.
2. Answer five guided interview questions.
3. The application collects your preferences.
4. A production-ready Claude system prompt is generated.
5. A specialized interface is created based on the selected use case.
6. Documentation is automatically included to explain the design decisions.
