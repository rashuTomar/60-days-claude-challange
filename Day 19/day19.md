# Day 19 – Football Intelligence Hub

## Overview

Built an AI-powered **Football Intelligence Hub** that delivers data-driven FIFA World Cup 2026 predictions, evaluates users' football knowledge through an interactive Football IQ quiz, and matches personality traits with football legends like Messi and Ronaldo.

## Key Features

* FIFA World Cup 2026 Prediction Report
* Adaptive Football IQ Assessment
* Messi vs Ronaldo Personality Analysis
* Personalized Football Intelligence Profile
* Knowledge-level based response customization
* Multi-stage conversational AI workflow

## Key Learnings

* Designed structured multi-stage AI prompts.
* Learned adaptive prompt engineering based on user expertise.
* Built data-driven prediction workflows.
* Combined analytics, quizzes, and personality assessment into a single AI experience.
* Improved prompt orchestration for engaging and interactive user journeys.

## Tech & Concepts

* Claude Custom Skills
* Prompt Engineering
* AI Workflow Design
* Conditional Logic
* Personality Analysis
* Interactive User Experience (UX)
* Structured Output Generation
