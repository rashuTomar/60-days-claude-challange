# 📄 PDF Splitter & Merger

A premium single-page HTML application for splitting and merging PDF files directly in the browser. Built with modern UI/UX principles, responsive layouts, and client-side PDF processing to ensure complete privacy.

## ✨ Features

### PDF Splitter

* Upload PDF files
* Automatic page detection
* Page thumbnail previews
* Split by page numbers
* Custom page ranges
* Split after specific pages
* Split every N pages
* Extract selected pages
* Multiple split operations in one session
* Input validation and preview

### PDF Merger

* Drag-and-drop PDF uploads
* Upload multiple PDF files
* Sortable file list
* Reorder PDFs before merging
* File previews and page counts
* Merge into a single PDF
* One-click download

## 🚀 Additional Features

* 100% Client-side processing
* No server uploads
* Privacy-first design
* Offline support (after initial load)
* Responsive interface
* Dark mode
* Smooth animations
* Accessibility support
* Loading indicators
* Error handling
* Keyboard shortcuts
* Modern commercial-grade UI/UX

## 🛠️ Technologies Used
