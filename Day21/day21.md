# Day 21 – Digital Privacy Intelligence Dashboard

## Overview

A modern, cybersecurity-inspired dashboard that visualizes a user's reported digital footprint. The application transforms a list of digital services into meaningful privacy analytics while clearly distinguishing **Facts** from **Estimates** to promote responsible AI practices.

## Features

* 📊 Digital Footprint Score
* 🔒 Privacy Score
* 🔥 Exposure Heatmap
* 🏢 Company Exposure Ranking
* 📋 Data Collection Matrix
* 🎯 Risk Radar Visualization
* 👤 Digital Twin Profile
* 💎 Most Valuable Data Assets
* 🚀 Privacy Improvement Simulator
* ✅ Personalized Privacy Improvement Plan
* 📑 Final Privacy Assessment

## Tech Stack

* HTML5
* CSS3
* Vanilla JavaScript
* Canvas API (Risk Radar Visualization)

## Key Learnings

* Designed a premium SaaS-style analytics dashboard.
* Built responsive UI components using pure HTML, CSS, and JavaScript.
* Visualized privacy and cybersecurity insights through interactive charts and scorecards.
* Applied responsible AI principles by separating factual information from inferred estimates.
* Improved dashboard design, information hierarchy, and user experience.

## Disclaimer

This dashboard is intended for educational purposes only. It analyzes only the user-provided dataset and does **not** access private databases, personal accounts, or hidden information. Any behavioral or lifestyle insights are presented as **Estimates** rather than facts.
