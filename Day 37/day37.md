# 🧭 Task Compass

**Task Compass** is an interactive management simulation that teaches how work flows through real organizations. Instead of focusing on job titles, players learn the principles of **ownership, delegation, collaboration, and workflow thinking** through realistic workplace scenarios.

## ✨ Features

* 🏢 Multiple workplace environments (Tech, Startup, Corporate, Healthcare, Retail, Education, and more)
* 🎮 Three interactive stages:

  * **Who Owns This?** – Identify the primary owner of workplace tasks.
  * **Task Routing** – Build the correct workflow using drag-and-drop.
  * **Collaboration Challenge** – Assign multiple teams to solve complex situations.
* 📊 Understanding dashboard with progress tracking
* 📈 Final reflection with personalized insights and learning summary
* 🎨 Premium dark glassmorphism UI with smooth animations
* 📱 Fully responsive and works on desktop, tablet, and mobile
* 💻 100% offline – built with HTML, CSS, and Vanilla JavaScript

## 🚀 Getting Started

1. Download or clone the repository.
2. Open `index.html` in any modern web browser.
3. Choose a workplace type and begin exploring how work moves through organizations.

No installation or internet connection is required.

## 🛠️ Tech Stack

* HTML5
* CSS3
* Vanilla JavaScript
