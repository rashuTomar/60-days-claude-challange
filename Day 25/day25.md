# 🦈 AI Shark Tank Simulator

An interactive **AI-powered Shark Tank Simulator** built as a **single self-contained HTML file**. The application allows users to pitch startup ideas, answer investor questions, receive AI-generated evaluations, and obtain simulated investment decisions in a modern, responsive interface.

## ✨ Features

* Startup pitch submission with structured business details
* Four AI judges with unique evaluation perspectives:

  * Venture Capitalist
  * Founder
  * Customer
  * Angel Investor
* Interactive pitch round with dynamic Q&A
* Multi-category startup scoring system
* AI-generated investment decisions
* Suggested valuation and funding recommendations
* Shark Tank-inspired dark theme UI
* Responsive design with smooth animations
* Confetti celebration for successful investments
* Downloadable Pitch Report (PDF)
* Leaderboard system
* Share result functionality

## 🛠️ Tech Stack

* HTML5
* CSS3
* Vanilla JavaScript
* Client-side PDF Generation
* Responsive UI Design

## 🎯 Scoring Parameters

* Market Potential
* Innovation
* Business Model
* Execution
* Investment Worthiness

## 💰 Possible Investment Outcomes

* ✅ Invest
* ❌ Reject
* 🤝 Acquire
* ⏳ Come Back Later

Each decision includes AI-generated reasoning, suggested valuation, and recommended funding amount.

## 📂 Project Structure

```
AI-Shark-Tank-Simulator/
│── index.html
```

## 🚀 How to Run

1. Download or clone the repository.
2. Open `index.html` in any modern web browser.
3. Enter your startup details.
4. Pitch your idea, answer the AI judges' questions, and receive your investment decision.
